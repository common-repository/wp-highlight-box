== Description ==

== Installation ==

1.Install plugin from Wordpress menu Plugins > Add New
2.Activate the plugin
3.Well done.
4.using:
[highlight-box class="wp-highlight-box-blue" style="background:#000000;color:#000000" content="Your content here"]

[highlight-box class="wp-highlight-box-green" style="background:#000000;color:#000000" content="Your content here"]
